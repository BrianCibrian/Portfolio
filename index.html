iSee

iSee is a web application designed to assist visually impaired individuals by identifying objects in their surroundings using voice-activated commands. Developed using HTML, CSS, and JavaScript, the project leverages the Web Speech API and TensorFlow Object Detection API to enable real-time object recognition through a device's camera. Created as a prototype and proof of concept for a class project on the education of those with disabilities, iSee demonstrates potential applications in smart glasses technology. The project received approval and positive feedback from our TA, who is visually impaired, highlighting its practical utility and innovation.

Teach Your Tots

Teach Your Tots was my final group project for an Introduction to Engineering class, aimed at helping children develop motor skills, speech, and hand-eye coordination. Coded in HTML, CSS, and JavaScript, our website features four mini-games: an alphabet game teaching letters, words, and sentences; a numbers game focusing on reading numbers and basic arithmetic; a shape-drawing game teaching shapes and requiring users to draw them; and a chatbot game allowing children to practice their skills in conversation. We utilized the Web Speech API for speech recognition and synthesis, and the p5.js library to enhance game interactions.

Around the World

This was my final project for the Google CSSI program, developed in collaboration with a partner. Coded in HTML, CSS, and JavaScript, we aimed to create a website for individuals planning to travel but unsure of their destination. With Around the World, users can select a continent and a landscape type, and the site generates suggested locations complete with embedded Google Maps and descriptions from TripAdvisor. Additionally, the site features a currency page that allows users to view exchange rates using an exchange rate API, providing comprehensive travel information in one place.

GifFinder

GifFinder is a website designed to entertain users by generating GIFs based on their input. Coded in HTML, CSS, and JavaScript, the site utilizes the Giphy API to fetch and display GIFs matching the user's query. Users simply enter a keyword, and GifFinder dynamically retrieves and showcases the relevant GIFs, providing an engaging and fun experience.
